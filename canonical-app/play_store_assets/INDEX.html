📋 How to Create Play Store Screenshots

  1. Open each HTML file in Google Chrome
  2. Press F12 → click the device toolbar icon (📱)
  3. Set dimensions to 1080 × 1920 in the device toolbar at top
  4. Press Ctrl+Shift+P → type "screenshot" → click "Capture full size screenshot"
  5. Save as PNG — ready to upload to Play Store!

📐 Play Store Screenshot Requirements

Format: JPEG or 24-bit PNG (no alpha)
Min size: 320px on shortest side
Max size: 3840px on longest side
Aspect ratio: max 2:1
Recommended: 1080×1920 (portrait) ✅ — these files are exactly that size

🎬 How to Create a Play Store Promo Video

1. Open DEMO_VIDEO_booking_flow.html in Chrome at 1080×1920
2. Use OBS Studio (free) or Windows Game Bar (Win+G) to screen record
3. Record for 30–90 seconds while the demo auto-plays
4. Export as MP4 — Play Store accepts MP4 up to 2 minutes
5. Upload in Play Console under "Promo Video"
